如何使用

在文件夹中创建一个scn文件夹，放入scn文件到其中

点击

1readscn.cmd

2initjson.cmd

3readjson.cmd

注意如果有新的json添加请重新

点击2initjson.cmd创建处理列表

然后再运行3readjson.cmd

本程序仅适用于nekopara ex/vol0,1,2,4,After

nekopara vol3的int中的cstl文本提取请到：

https://github.com/Kinotern/CatSystem2-Simple-Translating-Tools-Extra

感谢一位不知名的B站用户开源此解决方案，本项目

在此基础，针对nekopara的三语json结构适配
